# KW-checklist 

This project is for assessment purpose only from https://kw-checklist.docs.stoplight.io/

## Pre-requisite
1. Node.js >= v12.x.
2. MongoDB v3.6.3

## Next steps
- clone this repository
- ```yarn install```
- ```yarn test``` (optional)
- ```yarn dev```
- base url `http://localhost:1337/`
